###対応コンパイラ

* Visual C++ 12.0 以上

###ビルドするために必要なこと

####使用ライブラリ

* Boost 1.55 以上
* Google Test 1.7.0 以上

####環境変数

* `BOOST_DIR`...boostのディレクトリへのパス
* `GTEST_DIR`...google testのディレクトリへのパス

Visual Studio 2013でビルドする場合はVisual Studio 2013でビルドされたGoogle TestとBoostが必要です。
